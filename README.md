# My ZSH Config

## Install

Install oh-my-zsh:

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

Install Pure Promt:

```bash
npm install --global pure-prompt
```

Install zshrc:

```bash
git clone https://github.com/kenguru33/my-zsh-config.git ~/.config/zsh
```
